Agentic-based Anomaly Detection System for IP Core Network

MLOps Time Series 2025.03 - 현재

프로젝트 개요

KT Decision Intelligence Lab에서 IP 코어 네트워크의 이상 탐지를 위한 Agentic 기반 시스템을 개발하고 있습니다.

주요 성과

  • 대규모 데이터 처리: 전국 70개 코어 라우터에서 수집된 28만 개 시계열 데이터를 1분 단위로 처리
  • 자동화된 임계값 최적화: 네트워크 트래픽 이상 탐지를 위한 자동화된 임계값 최적화 알고리즘 배포
  • 일일 4억 개 이상 데이터 포인트 처리: 체계적인 이상 탐지 시스템 구축
  • Root-Cause Analysis 시간 80% 단축: 30분 → 4분으로 단축
  • MLOps 프레임워크 구축: Databricks와 GitHub 기반 커스텀 MLOps 프레임워크 개발

기술 스택

  • Databricks
  • Apache Spark
  • MLOps
  • Time Series Analysis
  • Anomaly Detection
  • Stateful Rollback Mechanism

주요 기능

  • 실시간 모니터링 메타데이터 보존 (drift detection, problem flags 등)
  • 모델, 코드, 중앙화된 설정 상수의 원활한 동기화
  • 버전 롤백 시 엔드투엔드 환경 일관성 유지
  • 자동화된 진단 워크플로우 및 자율 AI 추론 에이전트 지원

전략적 파트너십

Palantir (AIP) 및 Microsoft (Fabric)와의 고임팩트 PoC를 통해 자율 에이전트 진단 워크플로우 및 고처리량 실시간 스트리밍 파이프라인을 검증했습니다.
