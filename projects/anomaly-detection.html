Agentic-based Anomaly Detection System for IP Core Network

KT Decision Intelligence Lab MLOps AI Agent Time Series

프로젝트 개요

KT의 핵심 인프라인 전국 IP 코어 네트워크의 안정성을 확보하기 위한 **지능형 이상 탐지 및 자동 대응 시스템**입니다. 전국 70여 개 코어 라우터에서 발생하는 대규모 시계열 데이터를 실시간으로 분석하여, 네트워크 장애의 전조 증상을 포착하고 원인을 분석(RCA)하는 에이전트 기반 워크플로우를 구축했습니다.

280,000+ 실시간 감시 지표(분당)
4억+ 일일 데이터 처리량
86% ↓ RCA 시간 단축 (30분→4분)

핵심 해결 과제

  • 대규모 데이터 처리: 전국 코어 라우터에서 쏟아지는 방대한 데이터를 지연 없이 처리할 수 있는 고성능 파이프라인 필요
  • 관리 효율화: 수십만 개의 지표에 대해 수동으로 임계치를 설정하는 방식의 한계 극복
  • 신속한 장애 대응: 이상 발생 시 단순 알람을 넘어 구체적인 원인 지점(Root Cause)을 자율적으로 식별하는 에이전트 필요
Fig 1. Agentic-based Diagnosis Workflow

주요 기술 및 성과

1. 커스텀 MLOps 프레임워크 구축

Databricks와 GitHub Actions를 결합하여 코드 형상 관리부터 모델 배포까지 자동화된 파이프라인을 구축했습니다. 특히 **Stateful Rollback Mechanism**을 통해 버전 롤백 시에도 이전의 메타데이터(Drift 탐지 정보 등)가 유지되도록 설계하여 엔드투엔드 일관성을 확보했습니다.

2. Agentic RCA (Root Cause Analysis)

단순한 임계치 초과 알람이 아닌, 에이전트가 네트워크의 토폴로지를 기반으로 연관된 지표들을 자율적으로 탐색합니다. 이를 통해 엔지니어가 수동으로 분석하던 **RCA 시간을 30분에서 4분으로 약 86% 단축**했습니다.

3. 글로벌 기술 PoC 및 협력

Palantir(AIP) 및 Microsoft(Fabric)와의 전략적 PoC를 통해 고처리량 실시간 스트리밍 파이프라인과 자율 에이전트 진단 워크플로우의 효용성을 검증했습니다.

기술 스택

  • Platform: Databricks (Unified Data Analytics), Apache Spark
  • MLOps: GitHub Actions, MLflow, Stateful Rollback Logic
  • Algorithm: Time-series Anomaly Detection, Drift Detection, Autonomous Diagnostic Agents
  • Data: 전국 IP Core Network Metrics (SNMP, Telemetry 등)
