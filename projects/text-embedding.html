Text-Embedding Similarity Model for Shipbuilding Procurement

NLP Text Embedding 2022.12 - 2023.01

프로젝트 개요

PricewaterhouseCoopers Consulting Technology Group에서 국내 3대 조선소의 프로세스 혁신 프로젝트에 참여하여 텍스트 임베딩 유사도 모델을 개발했습니다.

주요 성과

  • 품목 분류 체계 통합: 국내 3대 조선소의 상이한 품목 분류 체계를 통합
  • 자동화 구현: 수작업 매핑을 자동화된 NLP 기반 로직으로 대체
  • 공급망 진단 효율화: 크로스 컴퍼니 조달 데이터 동기화 및 공급망 진단 스트림라인화

기술 스택

  • Natural Language Processing (NLP)
  • Text Embedding
  • Similarity Matching
  • Data Integration

프로젝트 내용

각 조선소마다 다른 품목 분류 체계를 사용하고 있어 수작업으로 매핑하던 작업을 텍스트 임베딩 유사도 모델을 통해 자동화하여 효율성을 크게 향상시켰습니다.
