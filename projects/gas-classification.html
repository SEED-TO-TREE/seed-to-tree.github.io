Transfer Learning-based Mixture Gas Classification

대한산업공학회지 게재 Deep Learning Transfer Learning 산학협력(태성환경연구소)

프로젝트 개요

본 프로젝트는 산업 현장에서 발생하는 독성 가스 및 악취의 원인을 실시간으로 파악하기 위한 혼합 가스 분류 시스템 개발을 목적으로 합니다. UNIST Data Analytics Lab에서 태성환경연구소와 협업하여 진행하였으며, 소량의 가스 센서 데이터로도 높은 분류 성능을 내기 위해 전이 학습(Transfer Learning)을 도입했습니다.

Fig 1. Image Mapping Process

핵심 도전 과제

  • 비선형적 특성: 혼합 가스는 개별 가스의 선형 결합이 아닌 화학 작용으로 인한 비선형적 센서 응답값을 가짐
  • 데이터 수집 비용: 실험실 환경에서 정교한 혼합 가스 데이터를 생성하는 데 막대한 비용과 시간이 소요됨
  • 고차원 시계열: 다수의 센서(GSA)에서 수집되는 복잡한 시계열 데이터의 특징 추출 필요

주요 방법론

1. 이미지 매핑 (Image Mapping)

다변량 시계열 데이터(300초 간격)를 2D 유사 이미지 행렬(640x480x3)로 변환. 각 센서 응답값에 고유 색상을 지정하여 고유한 "전자 지문" 형성.

2. ResNet-50 전이 학습

ImageNet으로 사전 학습된 ResNet-50 모델 활용. 특징 추출 레이어는 고정(Frozen)하고, 출력층(Classifier)만 타겟 클래스에 맞게 학습.

Fig 2. Model Architecture

실험 결과 및 데이터

논문에서는 총 11개의 센서 어레이(TGS2602, PID-AH2, SS2198 등)를 사용하였으며, 가우시안 노이즈($x^* = x \times (1+z)$)를 통한 데이터 증강으로 불균형 문제를 해결했습니다.

데이터셋 실험 설정 평균 정확도 (Accuracy)
UCI Public Data 무작위 가중치 초기화 63.21%
전이학습 + 데이터증강 (제안) 99.51%
실제 실험 데이터 무작위 가중치 초기화 63.48%
전이학습 + 데이터증강 (제안) 99.13%

학술적 의의

  • 특징 추출 자동화: 복잡한 전처리 과정 없이 이미지 변환만으로 혼합 가스의 미묘한 패턴 차이를 자동 추출함
  • 소량 데이터 극복: 전이 학습을 통해 데이터가 적은 도메인에서도 딥러닝 모델 적용 가능성을 입증
  • 실질적 활용도: 실제 환경 모니터링 시스템에서 악취 및 독성 가스 판별을 위한 기초 기술로 활용 가능
